# Programming Languages Library

Idiomatic starter examples for every major general-purpose and systems programming language.

| Language | File | Run command |
|---|---|---|
| Python | [`python/main.py`](./python/main.py) | `python python/main.py` |
| JavaScript | [`javascript/main.js`](./javascript/main.js) | `node javascript/main.js` |
| TypeScript | [`typescript/main.ts`](./typescript/main.ts) | `npx ts-node typescript/main.ts` |
| Rust | [`rust/main.rs`](./rust/main.rs) | `rustc rust/main.rs -o /tmp/rust_main && /tmp/rust_main` |
| Go | [`go/main.go`](./go/main.go) | `go run go/main.go` |
| Java | [`java/Main.java`](./java/Main.java) | `javac java/Main.java && java -cp java Main` |
| C++ | [`cpp/main.cpp`](./cpp/main.cpp) | `g++ -std=c++20 -O2 cpp/main.cpp -o /tmp/cpp_main && /tmp/cpp_main` |
| C# | [`csharp/main.cs`](./csharp/main.cs) | `dotnet script csharp/main.cs` |
| Ruby | [`ruby/main.rb`](./ruby/main.rb) | `ruby ruby/main.rb` |
| Swift | [`swift/main.swift`](./swift/main.swift) | `swift swift/main.swift` |
| Kotlin | [`kotlin/main.kt`](./kotlin/main.kt) | `kotlinc kotlin/main.kt -include-runtime -d /tmp/main.jar && java -jar /tmp/main.jar` |
| Scala | [`scala/main.scala`](./scala/main.scala) | `scala scala/main.scala` |
| Haskell | [`haskell/main.hs`](./haskell/main.hs) | `runghc haskell/main.hs` |
| R | [`r/main.R`](./r/main.R) | `Rscript r/main.R` |
| Julia | [`julia/main.jl`](./julia/main.jl) | `julia julia/main.jl` |
| PHP | [`php/main.php`](./php/main.php) | `php php/main.php` |
| Lua | [`lua/main.lua`](./lua/main.lua) | `lua lua/main.lua` |
| Elixir | [`elixir/main.exs`](./elixir/main.exs) | `elixir elixir/main.exs` |
| Zig | [`zig/main.zig`](./zig/main.zig) | `zig run zig/main.zig` |
| Shell | [`shell/main.sh`](./shell/main.sh) | `bash shell/main.sh` |

Each example demonstrates the language's distinctive features including:
- Type system and generics
- Object-oriented and/or functional patterns
- Concurrency / async primitives
- Error handling
- Standard-library idioms
